# Prototipo Pastillero Automatico

El siguiente codigo fue un prototipo encargado por un cliente y cuenta con dos partes:

- Pastillero
- Brazalete

Ambas partes se cargan en un Arduino Nano con un modulo bluetooth.

## Objetivo

El pastillero esta programado para que a determinada hora dispense una pastilla. En ese momento manda un mensaje al brazalete. El brazalete tiene unos leds y al recibir el mensaje estos leds se encienden en se√±al de aviso. El brazalete lo tiene colocado un usuario. De esta forma el usuario es alertado que debe tomar la pastilla.
